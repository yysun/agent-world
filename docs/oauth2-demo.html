Setup Instructions

To test OAuth2 flows:

  1. Set environment variable: OAUTH2_CLIENT_ID=your-client-id
  2. Set environment variable: OAUTH2_CLIENT_SECRET=your-client-secret (optional for PKCE)
  3. Configure OAuth2 provider with redirect URI:
  4. Enter your client ID above and click a provider button

API Endpoints

  • GET /api/oauth2/authorize - Start authorization flow
  • GET /api/oauth2/callback - Handle authorization callback
  • GET /api/oauth2/discovery - Provider discovery information
